Week 2 Notes

Variables

Const and let are block-scoped.
Const: variable will not be reassigned to another value (value won’t change).
Let: if variable will be assigned to another value.

Scope

Its value only exists inside the block they are declared.
Global scope: a variable declared outside a block.
Local scope: a variable declared inside a block and only available in that block.

Template Literals

A special type of string.

“” or ‘’ = ``

Ex: ` She said, “It’s me!” `

const age = 39;
`I will be ${ age + 1 } next year`;
<< 'I will be 40 next year'

  • Array.pop(): removes last one

    (I can add , & / between words inside the parenthesis method)
  • Array.join(): turns array into a string
  • Array.reverse(): reverses order
  • Array.sort(): sets in alphabetical order

Callback

A callback is a function passed into another function as an argument to be executed later.

Examples:

const message = function() {
`console.log("This message is shown after 3 seconds");
}
setTimeout(message, 3000);
Here the message function is a callback

Or

setTimeout(function() {
console.log("This message is shown after 3 seconds");
}, 3000);
Anonymous callback

Also, it is used when we need to call the functions in a different order than the one typed.
