Week 9 Notes

The Window Object

- Alert: Window.alert(‘Hello’);

- Confirm: window.confirm(‘Do you want to quit?’);

- Width and height: window.outerHeight; window.innerHeight;
Outer: content, padding, border, margin.
Inner: only content.

- Location object: window.location;

- Redirect: window.location.href =’https://apple.com’

- Reload: window.location.reload();

- Browser and operating system: Window.navigator;

- History: window.history;

- Invoke a function after a set amount of time: Window.setTimeout();

- Repeatedly invoke a function: Window.setInterval();

- Optimized animation: Window.requestAnimationFrame();

- Cookies can be used to store small pieces of information between requests using the document.cookie property.

HTML5 APIs

- Data attribute:
HTML:data-product-type=”” (data-anyname)
JS:.dataset.productType

- Get coordinates: Navigator.geolocation

- Web storage: localStorage.something

- Web Workers: allow processes to be run in the background, adding support for concurrency in JavaScript.
Example:
var myWorker = new Worker("my_task.js");
myWorker.addEventListener("message", function (oEvent) {
console.log("Called back by the worker!\n");
}, false);
myWorker.postMessage(""); // start the worker.

- Web socket: allows two-way communication with a server – also known as push messaging.
Example:
var socket= new WebSocket("ws://www.example.com/webSocketserver.php");}

- Multimedia: tag< audio src=”” controls> same with video.

- Progressive web application: small screen displays like an app
