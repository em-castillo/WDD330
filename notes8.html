CSS Transform

Transform: rotate(45deg);

Transform: scale(2)  to increase size by 2 or (0.5) decrease.
It can also scale one side like a rectangle -> scaleX(2) scaleY(2)

Transform: skewX(20deg) or skewY(20deg) -> like a diamond.

Transform: translate(50px, 100px) -> move element. First is X, second is Y.

Transform: matrix([scaleX], [skewY], [skewX], [scaleY] ,[translateX], [translateY])

CSS Transition

It needs a trigger to happen like hover.

Transition-timing-function: ease in, cubic-bezier(), linear,etc.

Transition-delay: 1s

Or Transition: all .5s linear 1s;

Drag and Drop

HTML: draggable=’true’ -> inside an element with a class.

  • 1. Set the draggable attribute on any HTML elements you’d like to be draggable.
  • 2. Add an event listener for the dragstart event on any draggable HTML elements.
  • 3. Add an event listener for the dragover and drop events on any elements that you want to have accept dropped items.
