Object

- A car has properties like weight and color, and methods like start and stop.
- All cars have the same properties, but the property values differ from car to car.
- All cars have the same methods, but the methods are performed at different times.
- Objects are variables too. But objects can contain many values.
Example:
var car = {type:"Fiat", model:"500", color:"white"};

Object methods: "this"

- Objects are usually created to represent entities of the real world.
- Functions that are stored in object properties are called “methods”.
- To access the object, a method can use the this keyword.

Example:
let user = {
name: "John",
age: 30,
sayHi() {
// "this" is the "current object"
alert(this.name);
}
};
user.sayHi(); // John

Object Properties

- The name:values pairs in JavaScript objects are called properties.
firstName: "John";

How to access an Object Property?

person.firstName
Or
person["firstName"]

Object Methods

- A method is a function stored as a property.
- Methods are actions that can be performed on objects.
Example:
fullName: function() {return this.firstName + " " + this.lastName;}

How to access an Object Method?

name = person.fullName();

Document Object Model (DOM)

- Can access, change, add, or delete all the elements of an HTML document.
- The innerHTML property can be used to insert raw HTML directly into the DOM.
- The document.getElementById(), document.getElementsByClassName(), document.getElementsByTagNames() and document.querySelector() can be used to access elements on a page.
Example:
document.getElementById("demo").innerHTML = "Hello World!";

Events

- Link between the web page and user interactions.
- HTML events are "things" that happen to HTML elements.
- When JavaScript is used in HTML pages, JavaScript can "react" on these events.
- An event listener is attached to an element, then invokes a callback function when the event occurs.
Example:
onclick="this.innerHTML = Date()"
Or
onclick="displayDate()"
